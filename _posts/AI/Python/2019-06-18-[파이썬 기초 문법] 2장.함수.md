### 8.main함수가 있는 방식으로 작성할 때의 장점
~~~ python
def add(num1, num2):
    return num1 + num2
def min(num1, num2):
    return num1 - num2
def mul(num1, num2):
    return num1 * num2
def div(num1, num2):
    return num1 / num2

def main():
    print(add(5, 3))
    print(min(5, 3))
    print(mul(5, 3))
    print(div(5, 3))

main()
~~~
###  main함수가 없다면 사칙연산의 결과를 알고 싶을 때마다 번거롭게 계속 4번의 print함수를 실행해야한다.
